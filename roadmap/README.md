| **5 Users > 500 Features** | Validate with real people, not assumptions |
| **Cut Scope Ruthlessly** | Delete before adding |

## Success Metrics

### MVP (Week 4)
- [ ] 5 users complete first session
- [ ] At least 1 user returns for session 2
- [ ] Zero critical bugs
- [ ] Setup takes < 5 minutes

### v1.0 (Week 16)
- [ ] 100+ active users
- [ ] 50% week-over-week retention
- [ ] Paid subscribers
- [ ] < 3 second response time
