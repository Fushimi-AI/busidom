## Overview

Create a guided onboarding experience that helps new users set up their account, configure the app, and understand key features.

## Requirements

### Functional
1. Account creation wizard
2. Business setup flow
3. Feature tour
4. First conversation prompt
5. Progress tracking

### Non-Functional
- Complete in < 5 minutes
- Skippable
- Resumable

---

## Onboarding Steps

```
Step 1: Welcome
├── Intro video/animation
└── Value proposition

Step 2: Account Setup
├── Email/password
└── Name (optional)

Step 3: Business Setup
├── Business name
├── Industry
├── Current stage
└── Main challenge

Step 4: AI Configuration
├── LLM provider selection
├── API key entry
└── Test connection

Step 5: Feature Tour
├── Chat interface
├── Context sidebar
├── Stage progression
└── Commands overview

Step 6: First Conversation
├── Suggested prompts
├── Real interaction
└── Success celebration
```
