## What is Business-OS?

Business-OS is an AI-first operating system for entrepreneurs that provides:

- **Opinionated AI Mentor**: Get stage-specific advice from an experienced founder
- **Automated Workflows**: Pre-built workflows for customer validation, MVP planning, and more
- **Semantic Memory**: AI remembers your business context and past conversations
- **Automation Scheduler**: Daily briefs, weekly reviews, and custom recurring tasks
- **Local-First**: All data stored locally in SQLite, complete privacy

# Installation Guide

## Requirements

- **Node.js**: v18+ (LTS recommended)
- **npm**: v9+ (comes with Node.js)
- **Operating System**: macOS, Linux, or Windows
- **Storage**: 100MB for app + database
- **API Keys**: OpenAI API key (required)

---

## Installation Methods

### Method 1: NPM (Recommended)

Install globally via npm:

\`\`\`bash
npm install -g business-os
\`\`\`

Verify installation:

\`\`\`bash
bos --version
# Output: business-os v1.0.0
\`\`\`

## Troubleshooting

### Command Not Found

If `bos` command is not found after installation:

1. Check npm global bin directory:
   \`\`\`bash
   npm bin -g
   \`\`\`

2. Add to PATH if needed:
   \`\`\`bash
   export PATH="$(npm bin -g):$PATH"
   \`\`\`

### Permission Errors

On macOS/Linux, you may need sudo:

\`\`\`bash
sudo npm install -g business-os
\`\`\`

Or configure npm to install globally without sudo:

\`\`\`bash
npm config set prefix ~/.npm-global
export PATH=~/.npm-global/bin:$PATH
\`\`\`

### Command Reference Generation

```typescript
// scripts/generate-command-docs.ts
import { Command } from 'commander';
import fs from 'fs/promises';
import path from 'path';

export async function generateCommandDocs(program: Command) {
  const commands = extractCommands(program);

  for (const cmd of commands) {
    const markdown = generateCommandMarkdown(cmd);
    const filePath = path.join(__dirname, '..', 'docs', 'commands', `${cmd.name}.md`);

    await fs.writeFile(filePath, markdown);
  }

  console.log(`Generated docs for ${commands.length} commands`);
}

function generateCommandMarkdown(cmd: any): string {
  return `# ${cmd.name}

${cmd.description}

## Usage

\`\`\`bash
${cmd.usage}
\`\`\`

## Options

${cmd.options.map(opt => `- \`${opt.flags}\` - ${opt.description}`).join('\n')}

## Examples

${cmd.examples.map(ex => `\`\`\`bash\n${ex}\n\`\`\``).join('\n\n')}
`;
}
```
