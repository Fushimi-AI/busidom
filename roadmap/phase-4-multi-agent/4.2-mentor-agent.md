## Overview

Implement the primary Mentor Agent that provides opinionated business guidance, challenges assumptions, and coordinates with specialist agents.

## Requirements

### Functional
1. Extends BaseAgent with mentor personality
2. Access to memory retrieval tool
3. Can delegate to specialist agents
4. Stage-aware guidance
5. Challenges weak ideas

### Non-Functional
- Primary interface for users
- Maintains personality consistency
- Coordinates other agents
