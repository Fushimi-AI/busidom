## Overview

Collapsible sidebar showing business context, allowing users to view and edit extracted information in real-time.

## Requirements

### Functional
1. Display business context
2. Edit context fields inline
3. Show extraction confidence
4. Context history
5. Collapsible sections

### Non-Functional
- Real-time sync with chat
- Non-intrusive design
- Responsive layout
