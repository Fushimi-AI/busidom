## Overview

Build a settings panel for configuring LLM provider, API keys, appearance, and behavior preferences.

## Requirements

### Functional
1. LLM provider selection
2. API key management (secure)
3. Model selection
4. Appearance settings (theme, font)
5. Behavior settings (streaming, memory)
6. Account management

### Non-Functional
- Secure key storage
- Settings sync (optional)
- Instant apply

---

## Settings Categories

```
General
├── LLM Provider (OpenAI, Kimi, Anthropic)
├── Model Selection
├── API Key (encrypted)
└── Temperature/Max Tokens

Appearance
├── Theme (light, dark, system)
├── Font Size
└── Chat density

Behavior
├── Enable streaming
├── Enable memory
├── Auto-save conversations
└── Debug mode

Account
├── Email
├── Subscription tier
├── Export data
└── Delete account
```
