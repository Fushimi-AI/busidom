## Overview

Build the desktop application shell using Tauri (Rust + Web) for a native, lightweight experience with the existing Node.js backend.

## Requirements

### Functional
1. Cross-platform (macOS, Windows, Linux)
2. Native window with menu bar
3. System tray support
4. Auto-updates
5. Deep linking

### Non-Functional
- < 50MB app size
- < 100MB memory usage
- Native performance
- Secure IPC

---

## Technical Stack

```
Tauri 2.0 (Rust backend)
├── React 18 + TypeScript (frontend)
├── Tailwind CSS (styling)
├── Node.js sidecar (backend)
└── SQLite (local cache)
```
